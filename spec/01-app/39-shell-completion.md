# Shell Completion — Tab Autocompletion

## Overview

Gitmap provides shell tab-completion for all three major shells:
PowerShell, Bash, and Zsh. Completions cover subcommand names,
repo slugs (for `cd`/`go`), group names, and per-command flags.

The completion scripts are generated by gitmap-v27 itself and installed
automatically via `gitmap-v27 setup`.

---

## What Gets Completed

| Context | Completed Values |
|---------|-----------------|
| `gitmap-v27 <tab>` | All 41+ subcommand names and aliases |
| `gitmap-v27 cd <tab>` | Repo slugs from database + `repos`, `set-default`, `clear-default` |
| `gitmap-v27 go <tab>` | Same as `cd` |
| `gitmap-v27 cd repos --group <tab>` | Group names from database |
| `gitmap-v27 pull <tab>` | Repo slugs from database |
| `gitmap-v27 exec --group <tab>` | Group names from database |
| `gitmap-v27 <cmd> --<tab>` | Flags for the active command |

---

## Architecture

### Completion Data Source

Completions that require dynamic data (repo slugs, group names)
invoke gitmap-v27 subcommands that output machine-readable lists:

```bash
gitmap-v27 completion --list-repos    # one slug per line
gitmap-v27 completion --list-groups   # one group name per line
gitmap-v27 completion --list-commands # one command per line
```

These hidden flags are fast — they open the DB, query, and print.
No formatting, no color, no banners.

### Completion Subcommand

```bash
gitmap-v27 completion <shell>
```

Prints the completion script for the specified shell to stdout.
The user can source it manually or let `setup` handle installation.

| Shell | Output |
|-------|--------|
| `powershell` | `Register-ArgumentCompleter` block |
| `bash` | Bash completion function + `complete -F` |
| `zsh` | `#compdef gitmap-v27` function |

### Setup Integration

`gitmap-v27 setup` detects the current shell and:

1. Generates the appropriate completion script.
2. Writes it to a well-known location.
3. Adds a source line to the user's profile if not already present.

| Shell | Script Location | Profile |
|-------|----------------|---------|
| PowerShell | `$env:APPDATA\gitmap-v27\completions.ps1` | `$PROFILE` |
| Bash | `~/.local/share/gitmap-v27/completions.bash` | `~/.bashrc` |
| Zsh | `~/.local/share/gitmap-v27/completions.zsh` | `~/.zshrc` |

Setup is idempotent — running it again does not duplicate entries.

## Generated Scripts

### PowerShell

```powershell
Register-ArgumentCompleter -CommandName gitmap-v27 -ScriptBlock {
    param($wordToComplete, $commandAst, $cursorPosition)
    $args = $commandAst.CommandElements | Select-Object -Skip 1
    $cmd = if ($args.Count -gt 0) { $args[0].ToString() } else { "" }

    if ($cmd -eq "cd" -or $cmd -eq "go") {
        $repos = gitmap-v27 completion --list-repos
        $repos += @("repos", "set-default", "clear-default")
        $repos | Where-Object { $_ -like "$wordToComplete*" } |
            ForEach-Object { [System.Management.Automation.CompletionResult]::new($_) }
        return
    }

    $commands = gitmap-v27 completion --list-commands
    $commands | Where-Object { $_ -like "$wordToComplete*" } |
        ForEach-Object { [System.Management.Automation.CompletionResult]::new($_) }
}
```

### Bash

```bash
_gitmap_completions() {
    local cur prev cmd
    cur="${COMP_WORDS[COMP_CWORD]}"
    prev="${COMP_WORDS[COMP_CWORD-1]}"
    cmd="${COMP_WORDS[1]}"

    if [[ ${COMP_CWORD} -eq 1 ]]; then
        COMPREPLY=($(compgen -W "$(gitmap-v27 completion --list-commands)" -- "$cur"))
        return
    fi

    case "$cmd" in
        cd|go)
            if [[ "$prev" == "--group" || "$prev" == "-g" ]]; then
                COMPREPLY=($(compgen -W "$(gitmap-v27 completion --list-groups)" -- "$cur"))
            else
                COMPREPLY=($(compgen -W "$(gitmap-v27 completion --list-repos) repos set-default clear-default" -- "$cur"))
            fi
            ;;
        pull)
            COMPREPLY=($(compgen -W "$(gitmap-v27 completion --list-repos)" -- "$cur"))
            ;;
        exec)
            if [[ "$prev" == "--group" ]]; then
                COMPREPLY=($(compgen -W "$(gitmap-v27 completion --list-groups)" -- "$cur"))
            fi
            ;;
        group)
            COMPREPLY=($(compgen -W "create add remove list show delete" -- "$cur"))
            ;;
    esac
}
complete -F _gitmap_completions gitmap-v27
```

### Zsh

```zsh
#compdef gitmap-v27

_gitmap() {
    local -a commands repos groups
    commands=($(gitmap-v27 completion --list-commands))

    if (( CURRENT == 2 )); then
        _describe 'command' commands
        return
    fi

    case "${words[2]}" in
        cd|go)
            if [[ "${words[CURRENT-1]}" == "--group" || "${words[CURRENT-1]}" == "-g" ]]; then
                groups=($(gitmap-v27 completion --list-groups))
                _describe 'group' groups
            else
                repos=($(gitmap-v27 completion --list-repos))
                repos+=(repos set-default clear-default)
                _describe 'repo' repos
            fi
            ;;
        pull)
            repos=($(gitmap-v27 completion --list-repos))
            _describe 'repo' repos
            ;;
        exec)
            if [[ "${words[CURRENT-1]}" == "--group" ]]; then
                groups=($(gitmap-v27 completion --list-groups))
                _describe 'group' groups
            fi
            ;;
        group)
            _describe 'subcommand' '(create add remove list show delete)'
            ;;
    esac
}

compdef _gitmap gitmap-v27
```

## Flags

| Flag | Description |
|------|-------------|
| `--list-repos` | Print repo slugs, one per line (machine use) |
| `--list-groups` | Print group names, one per line (machine use) |
| `--list-commands` | Print all command names and aliases, one per line |

---

## File Layout

| File | Purpose |
|------|---------|
| `constants/constants_completion.go` | Command names, messages, script paths |
| `cmd/completion.go` | Completion subcommand and list output handlers |
| `completion/completion.go` | Script generators for each shell |
| `completion/install.go` | Profile detection and source-line insertion |

---

## Constraints

- List flags output one value per line with no decoration.
- Generated scripts use `gitmap-v27 completion --list-*` for dynamic data.
- Setup writes to profile only if the source line is absent.
- All files under 200 lines, all functions 8–15 lines.
- PowerShell scripts use ASCII only (no em-dashes, no Unicode).
